Project — 01

KOSPI Stock Anomaly Detection

KOSPI 주가 이상 탐지 시스템

Isolation Forest KRX API Python Z-Score Scikit-learn
↗ GitHub

Built an unsupervised anomaly detection pipeline for KOSPI 100 stocks covering 2020–2024, processing over 120,000 data points via an automated KRX API scraping pipeline. Isolation Forest model is benchmarked against a Z-score baseline and validated against 25 curated market events.

End-to-end pipeline from KRX API data scraping through feature engineering, model training, and market event validation — fully automated and reproducible.
Anomaly signals showed 3–5× higher returns in the following 5 trading days, with a 70%+ confirmation rate against real market events including COVID-19 crash and rate hike cycles.
Evaluated with Precision / Recall / F1 metrics; Isolation Forest outperformed Z-score baseline on imbalanced financial time series.

Project — 02

M5 Demand Forecasting

M5 수요 예측 MVP

LightGBM Time Series Python Feature Eng. Walmart Data
↗ GitHub ↗ Report

MVP forecasting pipeline for the M5 Forecasting Competition dataset — 42 days of hierarchical demand prediction across 3,049 Walmart product series spanning 3 states and multiple store/category levels. Focused on practical accuracy using gradient boosting with lag features, rolling statistics, and calendar effects.

Hierarchical forecasting across product / store / state levels with LightGBM; lag features, rolling mean/std, and SNAP event indicators as core feature groups.
Recursive multi-step prediction for 28-day horizon; evaluated with WRMSSE metric aligned with M5 competition scoring.
Pipeline designed for real-world applicability — reproducible feature engineering, clean train/val split respecting time order, and modular code structure.
