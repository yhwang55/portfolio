Federal Reserve Economic Data Chatbot

Full-stack AI chatbot that answers questions about U.S. Federal Reserve economic data in natural language — returning numbers, policy context, and future projections simultaneously. Built as Project Lead Engineer; designed and implemented the entire pipeline from data ingestion to frontend deployment. Awarded 1st place at KCU 2026.

SEP end-to-end pipeline — directly crawled Fed PDF projections (not available via API), structured into CSV, injected into GPT prompts, and overlaid on charts as dotted forecast lines.
Dual-endpoint RAG architecture — /api/chat returns text answers from ChromaDB-retrieved Fed speeches (223 docs) and FOMC minutes; /api/chart returns Plotly chart data from FRED API in parallel.
Natural language → FRED ticker mapping via keyword logic (no LLM call) with auto chart-type switching — time-series queries render as line charts, relationship queries as scatter plots.
APScheduler + retry pipeline — daily 4AM auto-update from Fed RSS; custom retry logic resolved 202-chunk gap from embedding rate limits (429).

Project — 02

KOSPI Anomaly Detection & Macro Factor Attribution

KOSPI 이상 탐지 + 거시경제 인과 분석

Isolation ForestHMMSHAPOLS RegressionKRX APIFRED APITableauStreamlit
↗ GitHub ↗ Streamlit ↗ Tableau ↗ Report

Full-stack market intelligence system built on top of a KOSPI 100 anomaly detection pipeline. Extended Isolation Forest anomaly detection with Hidden Markov Model regime classification (bull/bear), OLS regression and SHAP-based macro factor attribution, and interactive Tableau Public + Streamlit dashboards — covering 1,500+ trading days from 2020 to 2026 across 5 macro indicators.

Market-wide anomaly detection across KOSPI 100 stocks using multivariate Isolation Forest on 6 engineered features — flags systemic stress when 10%+ of stocks are anomalous on the same day.
HMM Regime Detection on KOSPI index returns identifies bull/bear regimes; CUSUM change-point detection provides non-parametric cross-validation of regime transitions.
OLS + SHAP attribution identifies USD/KRW as the dominant macro driver of anomaly frequency (p<0.001); VIX and US 10Y yield also statistically significant.
Deployed on Streamlit Cloud + Tableau Public with 7 interactive tabs: Anomaly Detection, Feature Analysis, Cross-Stock Comparison, Market Event Validation, Macro Overview, Regime Analysis, Factor Attribution.

Project — 03

M5 Demand Forecasting

M5 수요 예측 MVP

LightGBMTime SeriesPythonFeature Eng.Walmart DataStreamlit
↗ GitHub ↗ Streamlit ↗ Report

MVP forecasting pipeline for the M5 Forecasting Competition dataset — 28-day hierarchical demand prediction across 30,490 Walmart SKUs spanning 3 states and multiple store/category levels. Focused on practical accuracy using gradient boosting with lag features, rolling statistics, and calendar effects.

Baseline → LightGBM → Prophet 3-stage pipeline; LightGBM achieves RMSE -23%, MAE -24% vs expanding mean baseline on Walmart 30,490 SKU dataset.
Price elasticity analysis via log-log OLS regression quantifies category-level demand sensitivity — HOBBIES (b = -1.34) vs FOODS (b = -0.72).
Pipeline designed for real-world applicability — temporal split respecting time order, modular feature engineering, and Streamlit dashboard MVP.
